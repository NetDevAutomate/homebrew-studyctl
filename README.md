# homebrew-studyctl

Homebrew tap for [studyctl](https://github.com/NetDevAutomate/socratic-study-mentor) — an AuDHD-aware study tool with AI Socratic mentoring.

## Install

```bash
brew install NetDevAutomate/studyctl/studyctl
```

Or tap first, then install:

```bash
brew tap NetDevAutomate/studyctl
brew install studyctl
```

## Get Started

```bash
studyctl setup    # Interactive configuration wizard
studyctl web      # Launch the study web app
studyctl --help   # See all commands
```

## Optional Extras

```bash
pip install 'studyctl[all]'   # Install all optional features
```

See the [main repo](https://github.com/NetDevAutomate/socratic-study-mentor) for full documentation.
